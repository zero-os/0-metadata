Download resource.

post

Removing files or directories

get /api/v1/root/{path:*}

Download resource.

  • Request
  • Response

URI Parameters

  • path:*: required (string)

Query Parameters

  • depth: required (integer - default: 1)

    When listing directories, how deep to go

  • inspect: required (boolean - default: false)

    If true and path points to a file, return the metadata only of the file otherwise return the content of the file as binary stream. This parameter is ignored when the path points to a resource.

HTTP status code 200

Body

Media type: binary/octet-stream

Type: object

Media type: application/json

Type: object

Properties
  • name: required (string)

    Name for the file or directory entry

  • size: (integer)

    Size of the file. For directory entries this property will always be 0.

  • modified: (integer)

    Epoch of the last modification time.

    • Files:
      • Creation time (post)
      • Or the last time the file content was set (put).
    • Directories:
      • Creation time (post)
      • Or the last time the directory was changed (put).
      • Or the last time an entry was added or removed
  • created: (integer)

    Epoch of the creation time

  • children: required (array of )
    • Files: Always null
    • Directories: List of full or partial Inode objects required: false
  • path: (string)

    Path of the Inode

  • type: required (one of file, dir)

    Inode type. "file" or "dir"

Examples:

directory:

{
  "name": "Geert",
  "type": "dir",
  "size": 0,
  "modified": 1517426795,
  "created": 153689194,
  "children": [
    {
      "name": "Lowie",
      "type": "file"
    },
    {
      "name": "Arno",
      "type": "file"
    }
  ],
  "path": "/belgium/flanders/berlare/Geert"
}

file:

{
  "name": "Geert",
  "type": "file",
  "size": 193,
  "modified": 1517426795,
  "created": 153689194,
  "path": "/belgium/flanders/berlare/Geert"
}

HTTP status code 404

path not found

delete /api/v1/root/{path:*}

Removing files or directories

  • Request
  • Response

URI Parameters

  • path:*: required (string)

Query Parameters

  • recursive: required (boolean)

    In case path points to a directory that has child entries, the directory will be removed along with all its children. In case path points to a file, this parameter will be ignored.

HTTP status code 204

Node deleted

HTTP status code 400

Path has children and recursive is false

HTTP status code 404

Path not found
